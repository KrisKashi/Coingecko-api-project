 # Coingecko API crypto price fetcher

# Descripton 
 Simple Project that uses the coingecko API to fetch cryptocurrency price data. Built to practice working with external API's, practice python fundementals and structuring projects.

# Features
- Fetches pricing data from coingecko
- Parses JSON data 
- Basic error handling

# Technology used
- Python
- CoinGecko API 
- Requests library 

# Installation
  1. clone the repository 
  2. install the dependencies 
  (pip install requests)
  3. run the script
